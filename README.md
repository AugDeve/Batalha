🎮Jogo Batalha(C# Windows Forms)

📌Sobre o projeto:

Este é um jogo simples desenvolvido em C# com Windows Forms,
onde um avião controla o movimento e dispara contra aviões inimigos.

🛠Tecnologias usadas:

C#
Wondows Forms
Visual Estudio

🚀Funcionalidades:

Movimento do avião
Disparo de projéteis
Aviões inimigos aleatorios
Sistema básico de colisão

🎯Objetivo
Destruir os aviões inimigos, e não permitir a passagem!
